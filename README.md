# Crawler2017
本项目是一个爬取链家地产杭州二手房信息的爬虫项目。

共分为两部分：
第一部分在Ip_Get_class.py文件中，这部分主要是从免费的代理IP网站中批量爬取代理IP，并迭代访问'http://www.whatismyip.com.tw/'网站以验证IP的有效性，并最终返回一个包含所有有效IP的列表。

第二部分在main.py文件中，编写了两个包含爬虫的线程，线程一负责在主页中将所有详情页的URL一一爬取出来，写入队列中。线程二负责从队列中取出URL，并爬取对应详情页的信息。
